## About

I build machine learning systems with a background in applied ML and backend development.

My recent work has made me increasingly interested in how models are trained, evaluated, and improved in practice. I am especially drawn to the systems side of ML workflows and am currently exploring LLM post-training, evaluation, and model optimization.

---

## Selected Work

### Bird Diversity Analysis Platform
Backend system for large-scale ecological data analysis.

- Designed REST APIs for geospatial queries
- Integrated external APIs (eBird) with local database systems
- Optimized query flows using custom Django QuerySets
- Focused on performance, correctness, and clean system design

### ML Pipeline System *(WIP)*
End-to-end machine learning workflow system.

- Built pipeline stages for ingestion, preprocessing, training, evaluation, and inference
- Designed for modular experimentation and iteration
- Exploring evaluation workflows and model improvement loops
- Serving as a foundation for future LLM-oriented systems work

### LLM Post-training Project *(Planned)*
Early exploration of post-training techniques for language models.

- Prompt evaluation and comparison
- Output quality analysis and consistency checks
- Lightweight optimization methods such as prompting and fine-tuning
- Focus on improving reliability and model behavior
